# devOS — Frontend

Next.js 15 app. The thing you actually see.

```bash
npm install
npm run dev   # http://localhost:3000
npm run build # make sure nothing's on fire before deploying
```

Entry point: `src/app/page.tsx`
App windows: `src/components/apps/`
OS shell: `src/components/os/`
State: `src/store/` (Zustand)

See the root `README.md` for the full picture.
